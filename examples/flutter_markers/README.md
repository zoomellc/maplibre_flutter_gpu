# Flutter markers example

Existing sample projecting Tokyo Tower and Skytree through
`MapLibreMapController.toScreenOffset`, then placing ordinary Flutter widgets
over the map.

Android (API 29+, `arm64-v8a` device or `x86_64` emulator):

```bash
cd examples/flutter_markers
flutter run -d <android-device-id>
```

Apple platforms:

```bash
# macOS
cd examples/flutter_markers
flutter run -d macos

# iOS Simulator alternative
flutter run
```
